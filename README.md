# CodeAlpha_EDA
# CodeAlpha Internship - Exploratory Data Analysis (EDA)

## Project Overview
This repository contains my submission for **Task 2: Exploratory Data Analysis** as part of my Data Analytics internship at CodeAlpha. The goal of this project is to explore transactional data structures, discover hidden spending patterns, and identify correlations to drive business insights.

## Key Insights Uncovered
* **Spending Distribution:** The average customer transaction is ~$19.78, with the majority of orders falling between a modest $10 and $20 range.
* **Loyalty Program Impact:** Box plot analysis indicates that enrolled loyalty program members tend to have higher median spending thresholds compared to non-members.
* **Tipping Behavior:** A distinct positive correlation was identified between the total bill amount and tip sizes, with the dinner shift yielding the highest-spending and highest-tipping clientele.

## Tools Used
* **Python** (Google Colab Environment)
* **Pandas** & **NumPy** for data manipulation
* **Matplotlib** & **Seaborn** for data visualization
